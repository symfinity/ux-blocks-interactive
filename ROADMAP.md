## Details

### 0.1.x — Initial LiveComponent widgets

Core widget foundation, Symfony UX LiveComponent integration, shared UX Blocks conventions

### 0.2.x — Widget catalog

Additional widgets, reusable patterns, expanded component coverage

### 0.3.x — Composition

Nested components, composition patterns, reusable layouts, slot improvements

### 0.4.x — Forms

Live form widgets, validation patterns, asynchronous interactions, form integration

### 0.5.x — States

Loading states, empty states, error states, optimistic updates, feedback patterns

### 0.6.x — Accessibility

Keyboard navigation, focus management, screen reader improvements, accessibility validation

### 0.7.x — Developer experience

CLI generators, documentation, examples, testing helpers, diagnostics

### 0.8.x — Ecosystem integration

Deeper integration with ux-blocks-form, ux-blocks-interactive, ui-kernel, shared conventions

### 0.9.x — Stabilization

API review, deprecation cleanup, migration guides, documentation polish, performance review

### 1.0.x — Stable API

Semver-stable components, extension points, documented customization model, long-term API stability
